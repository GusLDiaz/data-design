Use Case

  • persona: Ricky James

Pre Conditions

  • already has an account, logged in
  • familiar with the interface
  • very engaged with in-site culture

Post Conditions

  • able to view own comment on the image
  • able to view replies to comment
  • able to reply to comments

A user views an image and wants to comment, and potentially reply to any replies to the original comment

Interaction Flow

  1. Ricky views a image on Imgur.com, decides he wants to comment
  2. Imgur prompts him with a text entry field
  3. Ricky writes his comment; hits submit
  4. Imgur writes text to database, returns success token
